## MATERIAL MANAGEMENT (MM)

### BAPI_MATERIAL_SAVEDATA — Create/Change Material
Purpose: Create or update material master data
Key Parameters:
  HEADDATA:   MATNR (material number), MTYPE (material type)
  CLIENTDATA: MEINS (base unit), MATKL (material group)
  PLANTDATA:  WERKS (plant), BESKZ (procurement type)

Example:
  CALL FUNCTION 'BAPI_MATERIAL_SAVEDATA'
    EXPORTING headdata   = ls_headdata
              clientdata = ls_clientdata
    TABLES    returnmessages = lt_return.

### BAPI_MATERIAL_GET_DETAIL — Read Material Detail
  CALL FUNCTION 'BAPI_MATERIAL_GET_DETAIL'
    EXPORTING material    = lv_matnr
              plant       = lv_werks
    IMPORTING materialgeneral = ls_general
    TABLES    returnmessages  = lt_return.

### BAPI_GOODSMVT_CREATE — Post Goods Movement (GI/GR)
  CALL FUNCTION 'BAPI_GOODSMVT_CREATE'
    EXPORTING goodsmvt_header = ls_header
              goodsmvt_code   = ls_code
    TABLES    goodsmvt_item    = lt_items
              return           = lt_return.

## BAPI RETURN CODE CHECK PATTERN

  READ TABLE lt_return WITH KEY type = 'E' TRANSPORTING NO FIELDS.

    CALL FUNCTION 'BAPI_TRANSACTION_ROLLBACK'.
    "handle error
  ELSE.
    CALL FUNCTION 'BAPI_TRANSACTION_COMMIT' EXPORTING wait = 'X'.
  ENDIF.
