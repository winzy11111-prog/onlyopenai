## GENERAL LEDGER POSTING

### BAPI_ACC_DOCUMENT_POST — Post FI Document (Universal)
Purpose: Post any FI accounting document (replaces FB01 for automation)
Key Parameters:
  DOCUMENTHEADER: BUS_ACT (RFBU), USERNAME, COMP_CODE, DOC_DATE, PSTNG_DATE
  ACCOUNTGL:      ITEMNO_ACC, GL_ACCOUNT, COMP_CODE, DEBIT_CRE (S/H), CURRENCY, AMT_DOCCUR
  CURRENCYAMOUNT: CURRENCY, AMT_DOCCUR, AMT_BASE
  RETURN:         Messages

Example:
  DATA: ls_header   TYPE bapiache09,
        lt_gl       TYPE TABLE OF bapiacgl09,
        lt_curr     TYPE TABLE OF bapiaccr09,
        lt_return   TYPE TABLE OF bapiret2.

  ls_header-bus_act    = 'RFBU'.
  ls_header-username   = sy-uname.
  ls_header-comp_code  = '1000'.
  ls_header-doc_date   = sy-datum.
  ls_header-pstng_date = sy-datum.
  ls_header-doc_type   = 'SA'.

  CALL FUNCTION 'BAPI_ACC_DOCUMENT_POST'
    EXPORTING  documentheader = ls_header
    TABLES     accountgl      = lt_gl
               currencyamount = lt_curr
               return         = lt_return.

  IF NOT lt_return[] IS INITIAL.
    CALL FUNCTION 'BAPI_TRANSACTION_ROLLBACK'.
  ELSE.
    CALL FUNCTION 'BAPI_TRANSACTION_COMMIT' EXPORTING wait = 'X'.
  ENDIF.

### BAPI_INCOMINGINVOICE_CREATE — Post Vendor Invoice (MIRO equivalent)

    EXPORTING  headerdata    = ls_headerdata
    TABLES     itemdata      = lt_itemdata
               return        = lt_return

## FI DOCUMENT OPERATIONS

### BAPI_FI_DOCUMENT_GET — Get FI Document Details
  CALL FUNCTION 'BAPI_FI_DOCUMENT_GET'
    EXPORTING  bukrs = '1000'
               belnr = lv_belnr
               gjahr = lv_gjahr
    TABLES     documentheader = lt_header
               documentitems  = lt_items
               return         = lt_return.

### BAPI_FI_DOCUMENT_REVERSE — Reverse FI Document
  CALL FUNCTION 'BAPI_FI_DOCUMENT_REVERSE'
    EXPORTING  bukrs        = '1000'
               belnr        = lv_belnr
               gjahr        = lv_gjahr
               reversal_reason = '01'
               reversal_date   = sy-datum
    TABLES     return       = lt_return.

| FBL3N  | G/L Account Line Items |
| FBL1N  | Vendor Line Items |
| FBL5N  | Customer Line Items |
